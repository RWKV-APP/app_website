RWKV Chat 2.4.0(build 521) 版本已发布

🔄 最近更新

- [新增] 添加了并行推理功能, RWKV 现在可以同时生成 2-8 个答案了(在 Android 设备上, 我们暂时只支持高通骁龙芯片, 在其他平台, 我们暂时仅支持使用 GPU 进行并行推理)
- [新增] 为离线翻译添加的中英互译功能
- [新增] 使用文言文回答问题
- [新增] 在设置页面添加了性能测试功能
- [优化] 优化了多处 UI/UX
- [修复] 修复了多个小问题
