RWKV Chat 2.5.2(build 536) 版本已发布

🔄 最近更新

- [新增] 支持了最新的 RWKV7-G1a 1.5B 模型(仅 Android 高通骁龙平台)
- [修复] 修复了多轮对话的 penalty 计算
- [新增] 预设解码参数
- [新增] 俄语翻译 @x-0D
- [移除] 移除 qnn g1 2.9B 模型，等 g1a 2.9B 发布后加回来
- [优化] 实现了树状 state 管理，动态清理 state 内存占用
- [优化] 优化了高通平台 decode & batch decode 推理性能
- [优化] 优化了会话页面的 UI 展示
- [优化] 优化了 App 内的若干文案
- [修复] 修复了一些不该出现的重新 prefill 情况
